README: 프로젝트 사용 가이드

실행 방법
1. 'ranking_app-win.exe' 실행
   - 실행 파일을 실행합니다.
2. 브라우저에서 접속
   - 브라우저에서 http://localhost:5000 에 접속합니다.

파일 생성 및 관리
- 실행 파일과 동일한 위치에 data.csv 파일이 자동 생성됩니다.
  - 랭킹 관리를 따로 해야 하는 경우: 파일 이름을 변경하거나 위치를 이동시킬 수 있습니다.

============================

기능 소개

점수 입력 페이지 (/form)
- 페이지 URL: http://localhost:5000/form
- 입력 항목:
  - Name:
    - 한글만 입력 가능
    - 최대 4글자, 최소 글자 수 제한 없음
  - Score:
    - 0~999 사이의 숫자 입력 가능
- 버튼 기능:
  - Submit 버튼:
    - 입력한 이름과 점수를 data.csv 파일에 저장
    - /result 페이지에서 입력한 점수를 실시간으로 화면에 표시
  - Go to list 버튼:
    - 현재 화면에서 점수 결과 표시를 비활성화
    - /result 페이지 랭킹 리스트 화면으로 바로 이동

랭킹 목록 페이지 (/result)
- 페이지 URL: http://localhost:5000/result
- 페이지 기능:
  - data.csv 파일의 데이터를 기반으로 랭킹 리스트를 표시
  - /form 페이지에서 점수를 입력하면 해당 점수를 화면에 20초간 표시
  - 20초 경과 후 랭킹 리스트 화면으로 자동 전환
  - Go to list 버튼:
    - 20초를 기다리지 않고 랭킹 리스트 화면으로 즉시 이동 가능

기타 참고 사항
- 모든 데이터는 data.csv 파일에 저장되므로 파일 관리에 주의하십시오.
- 브라우저와 연결된 localhost:5000을 통해 모든 기능을 사용할 수 있습니다.
- 브라우저 새로고침 시 리소스 로딩으로 인해 화면이 자연스럽게 표시되지 않을 수 있습니다. 이 현상은 최초 리소스 로딩이 완료된 후 해결됩니다.
- 실행파일을 재기동하면 웹소켓 등의 환경 설정이 초기화될 수 있습니다. 이 경우, 페이지가 정상적으로 동작하려면 웹 브라우저에서 새로고침 후 사용해주시길 바랍니다.
